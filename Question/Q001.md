## ✅ **1주차 - 기본 구조 실습 문제**

### 📝 **문제 설명**
- 정수 `n`을 입력받고,
- 공백이 없는 문자열 `s`를 입력받아,
- `n`번 문자열 `s`를 출력하는 프로그램을 작성하세요.

단, `std::ios::sync_with_stdio(false);`와 `cin.tie(nullptr);`를 꼭 사용해서 **입출력 최적화**를 적용하세요.

---

### 📌 **입력 예시**
```
3
Hello
```

### 📌 **출력 예시**
```
Hello
Hello
Hello
```

---

### ✅ **추가 포인트**
- `endl` 대신 `\n`으로 속도 비교해보기
- `string` 사용해보고 `cout` 출력 연습
- 반복문 for 활용 연습
