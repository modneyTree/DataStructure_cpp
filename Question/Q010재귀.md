## ✅ **문제 1. 팩토리얼 계산 (기초)**
### 📌 설명:
정수 `N`이 주어지면 `N!`을 재귀로 계산해 출력하시오.

### 📌 입력 예시:
```
5
```
### 📌 출력 예시:
```
120
```
✅ **포인트:** `N! = N * (N-1)!`

---

## ✅ **문제 2. 1부터 N까지 합 (기초)**
### 📌 설명:
정수 `N`이 주어지면 `1 + 2 + ... + N`의 합을 재귀로 구해 출력하시오.

### 📌 입력 예시:
```
10
```
### 📌 출력 예시:
```
55
```

---

## ✅ **문제 3. 피보나치 수열 (기초)**
### 📌 설명:
`N`번째 피보나치 수를 재귀로 계산해 출력하시오.
- `fib(0) = 0`, `fib(1) = 1`
- `fib(N) = fib(N-1) + fib(N-2)`

### 📌 입력 예시:
```
6
```
### 📌 출력 예시:
```
8
```

## ✅ **문제 5. 하노이 탑 (심화)**
### 📌 설명:
하노이 탑 문제를 재귀로 풀고,
- 옮긴 횟수와
- 어떤 원반을 어디서 어디로 옮겼는지 출력하시오.

### 📌 입력 예시:
```
3
```
### 📌 출력 예시 (예시 중 일부):
```
7
1 1 3
2 1 2
1 3 2
...
```
✅ **포인트:**  
- 원판 N개를 1번 → 3번으로 이동
- 재귀 핵심 패턴 문제
